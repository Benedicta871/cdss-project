# 🧠 MediCore CDSS — Clinical Decision Support System

A lightweight **Clinical Decision Support System (CDSS)** built with vanilla JavaScript that simulates real-world hospital workflows including lab analysis, doctor interpretation, and patient risk stratification.

## 📌 Overview

This project simulates a simplified hospital decision support system used in healthcare environments to:

- Evaluate lab results using rule-based logic
- Classify patient risk levels (Normal, Moderate, High, Critical)
- Provide doctor-level clinical summaries
- Switch between Lab View and Doctor View
- Track patient-level activity logs

---

## ⚙️ Features

### 🧪 Lab Module
- Displays full laboratory results
- Color-coded severity indicators
- Real-time patient switching

### 🩺 Doctor Module
- Clinical summary view
- Risk stratification
- Decision support output

### 🧠 Clinical Engine
- Rule-based logic system
- Evaluates:
  - Glucose
  - Potassium
  - LDL
- Generates severity classification

### 📊 System Features
- Role-based UI (Lab / Doctor)
- Patient selection dashboard
- Activity audit trail
- Responsive layout

---

## 🧱 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Rule-based logic engine (custom built)

---

## 🧠 Architecture

- `clinicalEngine.js` → medical logic layer
- `uiRenderer.js` → UI rendering engine
- `app.js` → state controller
- `dataService.js` → patient data layer
